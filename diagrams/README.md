# Diyagramlar

## Görüntüleme Seçenekleri

### Seçenek 1 — Çevrimiçi (En kolay)
1. https://www.plantuml.com/plantuml/uml adresine git
2. `.puml` dosyasının içeriğini yapıştır
3. PNG olarak indir

### Seçenek 2 — VS Code
1. "PlantUML" eklentisini kur
2. `.puml` dosyasını aç
3. Alt+D ile önizle

### Seçenek 3 — Komut satırı
```bash
brew install plantuml
plantuml diagrams/*.puml
```
PNG dosyaları aynı klasöre üretilir.
